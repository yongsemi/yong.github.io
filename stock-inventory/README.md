# 股票庫存試算表（可編輯版）

完整複製自 股票庫存試算表分享 : 股票庫存，完全可編輯。

## 如何建立試算表

1. 前往 [Google Apps Script](https://script.google.com) → 新增專案
2. 刪除預設程式碼，貼入 `Code.gs` 全部內容
3. 執行 `createStockSpreadsheet()` → 授權 → 試算表自動建立
4. 執行 `setupDailyTrigger()` → 啟用每日 **13:30（台北時間）** 自動更新

## 工作表結構

| 位置 | 內容 |
|------|------|
| 列 1–5 | 大盤指數（台灣加權、道瓊、那斯達克、費城半導體） |
| 列 7–11 | 持股彙總：美股 / 台股 / 英股 / Total |
| H1–L17 | 甜甜圈圖「股票總資產」 |
| 列 18 | 提示：填寫藍色方塊 |
| 列 19 | 表頭（股票, 股價, 股數, 成本台幣, 成本美金, 均價, 損益, 報酬%, 市值, 今日漲跌%, 今日損益） |
| 列 20–50 | 持股資料（最多 31 檔） |
| 列 53–56 | 匯率（TWD / JPY / CNY） |

## 填寫說明

**藍色欄位（手動填入）：**
| 欄 | 填寫內容 | 範例 |
|----|---------|------|
| A 股票 | GOOGLEFINANCE 代號 | `VOO`、`TPE:0050`、`LON:VWRA` |
| C 股數 | 持有股數 | `1000` |
| D 成本(台幣) | 買進總成本（台幣） | `90000` |
| E 成本(美金) | 買進均價（外幣）；台股留空 | `316.69` |

**自動計算欄位：**
- **股價** — `GOOGLEFINANCE(代號,"price")`
- **市值** — 台股：`股價 × 股數`；外股：`股價 × 股數 × USD_TWD`
- **均價** — 台股：`成本台幣 ÷ 股數`；外股：`成本美金 ÷ 股數`
- **損益** — `市值 − 成本台幣`
- **報酬%** — `損益 ÷ 成本台幣`
- **今日損益** — `市值 × 今日漲跌%`

## 股票代號格式

| 市場 | 格式 | 範例 |
|------|------|------|
| 美股 | 直接輸入 ticker | `VOO`、`NVDA` |
| 台灣上市（TWSE） | `TPE:代號` | `TPE:0050`、`TPE:2330` |
| 台灣上櫃（OTC） | `TAI:代號` 或 `代號.TW` | `00679B.TW` |
| 英股（倫敦） | `LON:代號` | `LON:VWRA` |

> **冷門股提示：** 若 GOOGLEFINANCE 抓不到價格（顯示 `#N/A`），請在 B 欄手動輸入股價，或使用 IMPORTXML 抓取。

## 每日 13:30 自動更新

執行 `setupDailyTrigger()` 後，每天 13:30（台北時間）系統會自動重新整理股價，確保收盤後數據正確更新。觸發記錄可在 Apps Script → 執行記錄 查看。
